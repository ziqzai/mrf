Basic Troubleshoot Guide

💻 PC Tidak Boleh Hidup

1. Pastikan kabel kuasa disambungkan dengan betul.

2. Cuba tekan butang kuasa selama 10 saat untuk reset.

3. Periksa jika ada bau hangit atau bunyi pelik. Jika ada, hubungi IT support segera.

🖨️ Printer Tidak Boleh Print

1. Pastikan printer dihidupkan.

2. Periksa kabel USB atau sambungan Wi-Fi.

3. Restart printer dan cuba print semula.

4. Semak jika ada kertas yang tersangkut di dalam printer.

🌐 Internet Tidak Berfungsi

1. Periksa sambungan Wi-Fi di komputer.

2. Restart modem/router.

3. Cuba sambungkan ke rangkaian lain untuk pastikan isu pada rangkaian atau peranti.

4. Jika masalah berlarutan, hubungi IT support.

📧 Tidak Boleh Akses Email

1. Semak jika kata laluan yang dimasukkan adalah betul.

2. Cuba log masuk menggunakan peranti lain.

3. Pastikan sambungan internet stabil.

4. Jika akaun terkunci, hubungi IT support untuk reset.
